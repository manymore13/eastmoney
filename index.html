💻 试试这些命令

终端 / Command Prompt
① 查看有哪些行业
report list
# 运行后你会看到所有行业列表：游戏、医疗、新能源...
② 查询游戏行业的研报（看最近5篇）
report query -i 1046 -s 5
# -i 1046 是游戏行业的代码，-s 5 是要5篇
③ 想知道更多玩法？
report -h
# 查看所有命令和参数说明

🚀 3分钟快速入门

1

下载项目

打开终端，把项目克隆到本地：

git clone https://github.com/manymore13/eastmoney.git
2

安装依赖

进入文件夹，安装Python依赖：

cd eastmoney && pip install -r requirements.txt
3

开始使用！

运行下面的命令，看看效果：

report list

💡 常见用法示例

🎮 查看游戏行业研报
report query -i 1046 -s 10
查询游戏行业（代码1046）的最新10篇研报
📈 看策略报告
report query -t strategy -s 5
看券商对市场行情的分析报告
🍷 看茅台的研报
report query -t stock -c 600519
查询贵州茅台（600519）的个股研报
📥 下载PDF保存
report download -i 1046 -s 3 -o ./reports
下载游戏行业最新3篇研报的PDF
📊 导出到Excel
report query -i 1046 --save-csv
查询结果保存为CSV文件，可用Excel打开
🔎 搜索行业
report list -s 新能源
搜索包含"新能源"关键词的行业
