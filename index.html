快速上手

Terminal
# 查看所有行业
$ report list
# 查询游戏行业研报
$ report query -i 1046 # -i 指定行业代码
# 下载研报 PDF
$ report download -i 1046 -s 5 -o ./reports # -s 数量 -o 输出目录
# 查询策略报告
$ report query -t strategy -s 10
# 查询个股研报（如茅台）
$ report query -t stock -c 600519
