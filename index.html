MCP Server Quick Start
① Install package
pip install eastmoney-reports
② Add to MCP config (e.g., claude_desktop_config.json)
{ "mcpServers": { "eastmoney": { "command": "eastmoney-mcp" } } }
③ Start using in your AI agent
eastmoney-mcp
# Or use via prompt: "Query reports for gaming industry (code 1046)"

💻 CLI Quick Start

Terminal
Install
pip install eastmoney-reports
List industries
eastmoney list
Query reports
eastmoney query -i 1046 -s 5
# Gaming industry, 5 reports

Install

💡 Examples

🎮 Gaming Reports
eastmoney query -i 1046 -s 10
📈 Strategy Reports
eastmoney query -t strategy -s 5
🍷 Stock Reports
eastmoney query -t stock -c 600519
📥 Download PDF
eastmoney download -i 1046 -s 3 -o ./reports
