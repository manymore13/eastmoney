## 用户需求

开发一个友好的命令行工具，用于获取东方财富研报数据。

## 核心功能

1. **支持多种研报类型**：个股研报、行业研报、策略报告、宏观研究、券商晨报
2. **查询功能**：按行业查询研报，支持分页查询，查询结果展示关键内容（研报名称、日期、机构、评级等）
3. **下载功能**：可下载PDF研报，支持指定存放目录，按研报类型自动建立目录存放
4. **批量下载**：支持批量下载指定行业或股票的研报
5. **行业列表**：支持列出所有可选行业

## 数据字段要求

查询结果至少包含：研报标题、发布机构、发布时间、行业/股票、研报类型、研报链接

## 技术栈

- Python 3.x
- 内置argparse（CLI框架）
- requests（HTTP请求）
- lxml（HTML解析，用于提取PDF链接）
- asyncio（异步下载）

## 技术架构

### 系统架构

CLI工具采用分层架构：

- **入口层**：argparse命令行解析
- **业务层**：EastMoneyReport类，封装各类研报API
- **数据层**：CSV/JSON文件处理
- **下载层**：异步PDF下载

### API接口规划

根据东方财富现有API结构扩展：

- 行业研报：`industryCode`参数
- 个股研报：`stockCode`参数（新增）
- 策略报告：`qType=3`参数（新增）
- 宏观研究：`qType=4`参数（新增）
- 券商晨报：`qType=2`参数（新增）

### CLI命令设计

```
python -m eastmoney query --type industry --industry 1046 --page 1 --pagesize 20
python -m eastmoney query --type stock --code 600519 --page 1
python -m eastmoney query --type strategy --page 1
python -m eastmoney query --type macro --page 1
python -m eastmoney query --type morning --page 1

python -m eastmoney download --type industry --industry 1046 --output ./reports
python -m eastmoney download --type stock 600519 --output ./reports

python -m eastmoney list-industries
```
